## Run Locally

Clone the project

```bash
  git clone https://github.com/adonbiju/React-UI-Designs.git
```

Go to the project directory

```bash
  cd React-UI-Designs\gpt3_ui_design\
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm  start
```
