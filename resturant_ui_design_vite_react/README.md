## Run Locally

Clone the project

```bash
  git clone https://github.com/adonbiju/React-UI-Designs.git
```

Go to the project directory

```bash
  cd React-UI-Designs\resturant_ui_design_vite_react\
```

Install dependencies

```bash
  npm install
```

Start the porject

```bash
  npm  run dev
```
